# uniforms-bootstrap4

> bootstrap4 UI components for `uniforms`.

## Install

```sh
$ npm install uniforms-bootstrap4
```

For more in depth documentation see: [https://github.com/vazco/uniforms/](https://github.com/vazco/uniforms/).
